# idea-helper
立志于翻译 [IntelliJ IDEA 帮助文档](https://www.jetbrains.com/help/idea/meet-intellij-idea.html)，不定期更新。


## 关于IDEA
``IntelliJ IDEA`` 是 ``Java 语言`` 的开发神器，其 **智能提示**、 **自动补全**、 **代码重构**，以及对 ``J2EE`` 的支持，可以给开发者带来前所未有的开发体验，让开发者对于编程的感觉，就像在艺术创作一样。
